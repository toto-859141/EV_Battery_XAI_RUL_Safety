# Literature Review

This folder is the working space for collecting, organizing, and synthesizing references related to EV battery remaining useful life (RUL), safety risk prediction, and explainable AI (XAI).

## Goals

- Track the most relevant papers, datasets, and benchmark methods
- Identify research gaps for the proposal and manuscript
- Summarize modeling approaches, features, metrics, and explainability methods

## Suggested Contents

- `paper_log.md`: a running table or list of reviewed papers
- `themes.md`: grouped notes by topic such as RUL, thermal runaway, uncertainty, or XAI
- `references.bib`: citation library for writing

## Review Template

For each paper, capture:

- Citation
- Research objective
- Dataset used
- Input features
- Target variable
- Model type
- Evaluation metrics
- Explainability method
- Key findings
- Limitations
- Relevance to this project
